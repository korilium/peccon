The story of an aspiration

Intro

This page describes the thoughts and motivation behind the project peccon. It is subdivided into three main themes:

  • The Idea
  • The Journey
  • The end goal
The themes entail a thought process which makes this project viable. The story helps the builders organize all the necessary input on each theme and by consequence gives them a clear overview on what they want to achieve, how to get there and how it in the end should look like. The story is therefore not written for anyone but the builders. It is as it were a guideline to not lose track on what really matters.
